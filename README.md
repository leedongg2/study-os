# 🧠 Study OS

> 동규의 개인 학습·작업·휴식·경제 관리 시스템 (단일 HTML 앱)
> *"결정 피로 = 0. 시스템이 결정하고, 나는 실행한다. 그리고 제대로 쉰다."*

KSA(한국과학영재학교) 준비와 PMI 사업 운영을 병행하는 14세 솔로 파운더를 위한 **올인원 생산성 OS**. 외부 의존성 없이 단일 `study-os.html` 파일로 동작한다.

## ✨ 핵심 기능

| 카테고리 | 내용 |
|---|---|
| 🎯 **지금** | 결정 엔진(컨디션·시간·상황 → 즉시 최적 모드 추천) · NOW 위젯 · 유혹 방어(SHIELD/PANIC) |
| 📚 **공부** | 집중 타이머 + 포모도로 · 학습 과학 · 양자컴퓨팅/수학 강의(Shor, QFT 등) |
| 🏢 **PMI** | 사업/개발 작업 시간 추적 (TrapNet·PMC·CLI·Pick-My-AI) |
| 🌿 **휴식** | 회복형 휴식 관리 · 수면 추적 |
| 💰 **경제** | 코인 RPG · 🏦저축(복리이자) · 💳대출 · 🎲자기베팅 · 🏆재무등급 · 📊수입지출 그래프 |
| 🎮 **게임** | 빙고 · PVP · 트로피 |
| 📊 **분석** | 통계 · Brain 지능 엔진 · 히트맵 |
| 🌌 **목표** | 매크로 목표 · 비전 보드 |

## 🔒 기록 게이트
공부/작업을 **기록하지 않으면 게임·상점·베팅이 잠긴다**. 기록이 곧 게임의 입장료 (Premack 원리).

## 🔄 실시간 동기화
Supabase Realtime으로 **폰 ↔ 노트북 데이터가 새로고침 없이 실시간 연동**된다.

## 🛠 기술 스택
- 순수 HTML/CSS/JS (빌드 불필요, 단일 파일)
- localStorage (로컬 저장) + Supabase (클라우드 동기화)
- IBM Plex Mono + Gowun Batang · 다크 "Mission Control" 테마
